## Features

| | |
|---|---|
| **Live Discord Presence** | Polls the Lanyard REST API every 5 seconds to display real-time status, current activity, and active Spotify track. |
| **GitHub Friend Cards** | Fetches GitHub user profiles via the API and caches results in localStorage with a 1-hour expiry. |
| **Feedback Form** | Collapsible form that sends messages to a Discord webhook. Can be toggled on or off via site configuration. |
| **Scroll-Reveal Animations** | Intersection Observer-driven fade and slide-up effects with staggered delays. Respects `prefers-reduced-motion`. |
| **Animated Footer** | Canvas-rendered metallic paint logo with continuous animation. |
| **Page Loader** | Overlay with status-specific messaging (assets, Discord presence, project data). Auto-dismisses on load complete. |
| **SEO** | JSON-LD structured data (Person, WebSite, CollectionPage, BreadcrumbList), Open Graph and Twitter card meta tags. |
| **Accessibility** | Semantic HTML landmarks, ARIA attributes on interactive elements, reduced motion support, visible focus states. |
| **Content Protection** | Disables text selection, right-click context menu, and drag-to-save on images. |

---

## Stack

| Layer | Detail |
|---|---|
| Markup | HTML5 -- semantic, hand-written, landmark elements |
| Styling | CSS3 -- custom properties, flexbox, grid, gradient backgrounds, 300+ lines of animation |
| Typography | Outfit (headings) + Space Grotesk (body) served via Google Fonts with preconnect hints |
| JavaScript | Vanilla ES6+ -- no frameworks, no bundlers, no runtime dependencies |
| Icons | Inline SVGs for zero network overhead |
| Dev server | Pure Node.js -- `http`, `fs`, `path` modules only, zero npm dependencies |

## Getting Started

```bash
git clone https://github.com/bencodess/benslogs.git
cd benslogs
node dev-server.mjs
```

The development server starts on `http://localhost:4173`. It serves static files with correct MIME types, protects against directory traversal, redirects directory requests to trailing slashes, and falls back to a custom 404 page.
